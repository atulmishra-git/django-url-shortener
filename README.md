# django-url-shortener
> A URL Shortener application in Django

### Install
```
pip install -r requirements.txt
```

### Run
```
> python manage.py makemigrations
> python manage.py migrate
> python manage.py runserver
```

### Run Test
```
python manage.py runserver
```
